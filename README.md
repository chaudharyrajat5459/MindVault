# 🧠 MindVault - Notes Management System

MindVault is a modern and responsive Notes Management web application built using **HTML, CSS, JavaScript, Node.js, and Express.js**. The application allows users to create, save, organize, and manage notes efficiently through a clean and user-friendly interface.

Designed with simplicity and productivity in mind, MindVault provides a seamless note-taking experience while demonstrating full-stack web development concepts such as frontend design, backend integration, data handling, and CRUD operations.

## ⚙️ How It Works

1. User creates a new note through the interface.
2. Note data is sent to the Express.js server.
3. The backend processes and stores the note in a JSON file.
4. Saved notes are dynamically displayed on the dashboard.
5. Users can edit or delete notes as needed.
6. Changes are automatically reflected in the stored data.
